HTML-код списків

Списки використовуються для відображення швидких посилань у футері сторінки.

<ul>
  <li><a href="#">Головна</a></li>
  <li><a href="#">Каталог</a></li>
  <li><a href="#">Контакти</a></li>
</ul>

Теорія про теги та атрибути списків

Списки в HTML створюються за допомогою тегів <ul> (неупорядковані списки) та <ol> (упорядковані списки). Елементи списків позначаються тегом <li>. Для стилізації можна використовувати атрибути або CSS-класи.
