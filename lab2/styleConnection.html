Способи підключення стилів у HTML

1. Зовнішній файл CSS

Цей спосіб передбачає підключення окремого файлу зі стилями за допомогою тега <link>. Стилі, що містяться в окремому файлі, застосовуються до всіх сторінок, які його підключають.

2. Внутрішні стилі (Internal CSS)

Внутрішні стилі додаються безпосередньо в HTML-документ всередині тега <style> у секції <head>. Вони застосовуються тільки до цієї сторінки.

3. Inline-стилі

Inline-стилі задаються безпосередньо в атрибуті <style> елемента. Цей спосіб застосовується для окремих елементів, але його не рекомендується використовувати часто через складність підтримки коду.

4. Підключення CSS через JavaScript

Цей спосіб дозволяє динамічно додавати стилі або підключати CSS-файли за допомогою JavaScript. Використовується для адаптивних і інтерактивних вебсторінок.
